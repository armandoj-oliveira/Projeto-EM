Projeto - CSharp - Escolar Manager 2022

RESUMO:

Um estudante precisa de uma calculadora para fazer suas tarefas de matemática. Porém, por falta de uma o mesmo vai até uma papelaria
e ao adquirir a calculadora realiza os cálculos. A calcudora avançada consegue armazenar os valores primitivos
e fazer uma busca entre os mais procurados, ou, dos que estão dentre os parêmtros que o estudante esocolher.

- Uso de VAR (possibilidade do sistema optar por qual tipo primitivo a variável seja); - OK
- USAR LINQ (busca pelo números mais recentes usados pelo usuário e resultador); - OK
- USAR DELEGATES (chamada de método); - OK
- POO (usar para a Pessoa); - OK
- CONVERT (converter tipos primitivos); - OK
- UNBOXING E BOXING? (planejando);
- UML e USE CASE (documentação necessária);
- Clean Code, SOLID e Refatoração (mostrar prints). - OK
